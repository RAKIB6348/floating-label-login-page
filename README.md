# Floating Label Login Page

A clean and responsive login page with modern floating label input fields, built using pure HTML and CSS.

## Features

- Responsive login form
- Floating label input animation
- Modern card-based UI
- Email and password fields
- Pure HTML and CSS

## Technologies Used

- HTML5
- CSS3

## Preview

This project contains a simple `index.html` file that displays a modern floating label login page.
